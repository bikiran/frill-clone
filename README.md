# Frill Clone MVP - Ready to Launch! 🚀

A complete customer feedback platform built in Next.js + Supabase, ready to deploy this weekend.

## What You Get

✅ **Ideas/Feedback Board** - Users submit & vote on feature requests  
✅ **Public Roadmap** - Visual product roadmap (Planned → In Progress → Shipped)  
✅ **Announcements** - Post product updates & changelog  
✅ **Admin Dashboard** - Manage ideas & announcements  
✅ **Authentication** - Secure user accounts with Supabase Auth  
✅ **Real-time Updates** - Changes appear instantly across all users  

## Quick Start

### 1. Set Up Supabase (15 min)
- Sign up at https://supabase.com (free tier)
- Create project, copy credentials to `.env.local`
- Run `DATABASE_SETUP.sql` in SQL Editor

### 2. Run Locally (5 min)
```bash
npm install
npm run dev
```
Visit http://localhost:3000

### 3. Deploy to Vercel (5 min)
- Push to GitHub
- Connect to Vercel
- Add environment variables
- Deploy!
